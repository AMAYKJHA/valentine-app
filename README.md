# Valentine Quiz ❤️

A fun couples quiz app built with React and Vite. Two people enter their names, answer 5 questions about each other, and see how many answers match. Simple state management with useState, no external libraries.

To run: `npm install` then `npm run dev`. Open http://localhost:5173 in your browser.
